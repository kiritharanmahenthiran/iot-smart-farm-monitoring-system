# 🌾 Smart Farm Monitoring System (ESP32 + Blynk)

## 📌 Project Overview
This project monitors:
- Soil Moisture
- Rain Detection
- Temperature & Humidity

The data is sent to the Blynk cloud and monitored using a mobile phone.

---

## 🛠 Components Used
- ESP32
- Soil Moisture Sensor
- Rain Sensor
- DHT11
- 3D Printed Enclosure

---

## 📡 Working Principle
Sensors → ESP32 → WiFi → Blynk Cloud → Mobile App

---

## 📱 Mobile Monitoring
Real-time data is viewed using the Blynk IoT platform.

---

## 🧱 3D Printed Enclosure
Custom-designed enclosure to protect electronics.

---
